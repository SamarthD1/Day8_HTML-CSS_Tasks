Task 1: Simple Contact Form

Objective: Create a basic contact form with HTML5 input types.

Create a form with:

  • Name field
  • Email field
  • Message textarea
  • Submit button

Goal: Learn the basic structure of a form, including label, input, textarea, and button elements.




Task 2: Form with HTML5 Validation

Objective: Use HTML5 attributes like required, pattern, and type to validate form inputs.

Create a form with:

  • Username (text input)
  • Password (password input with min-length validation)
  • Email (email input)

Use pattern for custom password validation (e.g., password must include at least one uppercase letter).

Goal: Learn how to use HTML5 validation for better form control.




Task 3: Create a Multi-Page Form with fieldset

Objective: Use the fieldset element to group related form fields.

Create a form with 3 sections (Personal Information, Address Information, and Contact Details).

Use fieldset and legend to label each section.

Goal: Improve form organization using fieldset and legend.

Task 4: Form with Custom Radio Buttons and Checkboxes

  • Create a form with:
    • Multiple choices for the user to select (radio buttons)
    • A list of items the user can choose from (checkboxes)
  • Use CSS to style these form elements to look unique.

Goal: Learn how to use CSS for custom form element styling.











Task 5: Responsive Form Design

Objective: Make the form responsive using CSS media queries.

  • Create a basic form with fields such as Name, Email, and Message.
  • Add media queries to make sure the form looks good on **mobile devices** (e.g., stack fields vertically, adjust button size).

Goal: Learn how to use media queries to ensure forms are mobile-friendly.
